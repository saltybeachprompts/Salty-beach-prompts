
# 🌊 Salty Beach Prompts

AI-powered image transformation app that turns users into GTA, anime, Disney-style, and celebrity-inspired visuals.

## Features
- Image upload
- Style selection
- Stripe payments ($0.99 per use)
- AI transformation (coming soon)

## Setup

```bash
npm install
npm run dev
```

## Deploy
Use Vercel and add environment variables:
- STRIPE_SECRET_KEY
- NEXT_PUBLIC_BASE_URL
